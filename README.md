📌 Project Title

Smoking Health Risk Analysis using Python

📖 Overview

This project analyzes the impact of smoking behavior on various health indicators such as blood pressure, cholesterol levels, and organ-related conditions using Python-based data analysis and visualization techniques.

🎯 Objective

The objective of this project is to identify patterns and relationships between smoking habits and health risks to support better healthcare insights and decision-making.

🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab
📊 Dataset Description

The dataset includes patient-level information such as:

Age
Gender
Smoking Status
Cigarettes Per Day
Blood Pressure Risk
Cholesterol Level
BMI
Organ and Organ Conditions
🔍 Analysis Performed
Data loading and exploration
Data validation (checked for missing values)
Group-based analysis using Pandas
Comparative analysis across smoking categories
Visualization of key health indicators
📈 Key Visualizations
Organ vs Smoking Intensity
Smoking Status vs BP Risk
Smoking vs Cholesterol Levels
Age Distribution across Smoking Groups
💡 Key Insights
Smoking is associated with higher blood pressure risk levels
Smokers tend to have higher cholesterol levels
Certain organs show higher smoking exposure patterns
Smoking behavior varies across different age groups
🧠 Conclusion

This analysis helps in identifying high-risk individuals based on smoking behavior and supports better healthcare decision-making through early intervention and preventive strategies.
